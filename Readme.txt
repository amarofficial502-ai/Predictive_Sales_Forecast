## Project Scope

This project focuses on analyzing historical sales data to identify sales patterns and predict future revenue. Data cleaning, visualization, exploratory analysis, feature engineering, and machine learning techniques are used to build accurate sales prediction models.

- Pandas

Result

Random Forest Regressor Accuracy (R² Score): **99.94%**

Conclusion

The project successfully predicted future sales revenue using machine learning. Random Forest Regressor outperformed Linear Regression and produced highly accurate predictions, helping improve business forecasting and decision-making.
